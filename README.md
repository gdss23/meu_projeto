📱 Meu Projeto Mobile
Este projeto foi 100% escrito, estruturado e enviado para o GitHub usando apenas o celular (Acode).

🛠️ Como funciona
O arquivo principal é o app.py.
Ele foi feito para ser rodado no terminal do PC ou em servidores na nuvem depois de baixado do GitHub.
🚀 Como rodar depois
Abra o site do GitHub no PC ou celular
Baixe o arquivo app.py
Rode ele usando o Python instalado na máquina
Feito com o Acode no Android 🤖
